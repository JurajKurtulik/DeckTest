# Fabric Data Ingestion Presentation Site

Static interactive website for **Fabric in 6 Weeks - Week 3: Data Ingestion**.

## Use

Open `index.html` directly, or serve the folder with any static web server.

Keyboard controls:

- `Right` / `Down` / `Space`: next section
- `Left` / `Up`: previous section
- `Home`: first section
- `End`: final section

## GitHub Pages

This folder is ready for GitHub Pages without a build step.

Recommended setup:

1. Create a GitHub repository.
2. Commit the contents of this folder to the repository root.
3. In GitHub, open **Settings -> Pages**.
4. Set **Source** to `Deploy from a branch`.
5. Choose the `main` branch and `/root`.

The `.nojekyll` file is included so GitHub Pages serves assets exactly as written.

## Content Notes

The site preserves the deck sequence and terminology while combining related slides into web-native full-screen sections. Product icons and the Fabric mark are reused from the source deck assets; no fake product logos were introduced.
